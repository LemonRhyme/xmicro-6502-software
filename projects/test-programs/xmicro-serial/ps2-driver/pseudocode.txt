Interrupt

	If PA/BR flag set
		Read byte, decrement PA/BR index
		If index = 0, clear flag

	Else if Reset flag set
		If code =$AA, clear reset flag
		Else halt CPU (keyboard error)
			This part is for testing only.

	Else read code
		If code = $E0, set extended flag, return
		If code = $F0, set break flag, return
		If code = $E1, set PA/BR flag, reset other flags, set PA/BR index to 7 remaining bytes

		If code <= $A0, take the code. Else ignore it.
			Select a lookup table depending on state of extended flag
			Make/break the key depending on state of break flag
			Convert to keycode
				Store indexed in key buffer



READ FLAGS: %RPEB0000 - RESET, PABR, EXT, BRK

KEYCODE FLAGS:	%BSCAUG00 - BREAK, SHIFT, CTRL, ALT, SUPER, GUI
